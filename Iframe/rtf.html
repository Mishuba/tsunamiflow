Coverage: paragraph alignment (\qc \qj \ql \qr), first-line indent (\fi), font size (\fs), bold/italic/underline/strike/superscript/subscript/small-caps, \line and \par breaks, \page / \pagebb, skips font tables / stylesheets / doc info groups, basic \'hh hex-escape and \u unicode escapes, and \~ \- \_ special chars.

Not handled: tables, lists, embedded images/objects, colors, multiple fonts per run, revision marks, and RTF's more obscure destination groups. Real-world RTF from Word can use any of these — treat this as a solid starting point for simple documents, not a drop-in replacement for a full parser.
